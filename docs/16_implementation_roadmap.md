## 1. エグゼクティブサマリー

TsunaguはSMB向け採用自動化SaaSとして、AIスコアリング・マルチチャネル通知・成果報酬決済を統合したプラットフォームである。単独Founder体制で約4ヶ月後の2026-03-31にMVPリリースを目指す。

### 1.1 戦略的優先順位
- **MVP成功**が最優先：Stripe審査とLINE審査の遅延を許容しない
- **技術的負債回避**：PoC→本実装の段階的アプローチ
- **キャッシュフロー重視**：決済機能の早期完成を優先
